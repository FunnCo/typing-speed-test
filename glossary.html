Справка и термины

К тесту Статистика
Термины
  • WPM (std) — стандартный показатель скорости печати: (charsTyped / 5) / minutes, где charsTyped — число набранных символов (с учётом пробелов между отправленными словами), minutes — длительность теста в минутах.
  • WPM (слов) — скорость по словам: wordsCorrect / minutes.
  • CW — число верно набранных слов за попытку.
  • WS — число отправленных слов (всех подтверждённых пробелом/Enter).
  • Точность по словам — floor((CW / WS) × 100).
  • Точность по символам — процент совпадающих позиций символов при сравнении введённого слова с эталоном. Для каждого отправленного слова берётся max(len(typed), len(target)) позиций; отсутствующие/лишние символы считаются ошибкой.
  • Режимы:
    • база — слова из массива basicWords.
    • про — слова из массива topWords.
  • Лимит времени — 30 или 60 сек. В новой версии WPM (std) не использует множитель: всё считается по стандартной формуле.
Правила работы теста
  • Таймер стартует при первом вводе символа.
  • Пробел или Enter отправляет текущее слово на проверку.
  • Подсветка букв внутри текущего слова показывает ошибки мгновенно.
  • После окончания времени ввод блокируется, попытка сохраняется в localStorage и доступна на странице «Статистика».
Советы по набору
  • Смотрите на экран, а не на клавиатуру.
  • Высокая точность почти всегда улучшает итоговый WPM (std).
  • Тренируйтесь регулярно и сравнивайте прогресс в статистике (фильтры по режимам помогают).
О проекте

Цель — тренировка скорости и точности печати. Сайт реализован на HTML5, CSS3, JavaScript. Для адаптивной верстки используется Bootstrap 5, для визуализации статистики — Chart.js. Попытки сохраняются локально в localStorage.
